# InfoGraphIQ
Objective
To demonstrate an end-to-end data analytics pipeline, transforming raw sales data into actionable business insights using data wrangling, exploratory analysis, machine learning, and an interactive dashboard.

Tech Stack
Data Manipulation: Pandas, NumPy

Data Visualization: Matplotlib, Seaborn, Plotly

Machine Learning: Scikit-learn

Interactive Dashboard: Plotly Dash

Environment: Python 3.12, Jupyter Notebook / Google Colab

Requirements
To run this project, you will need all libraries listed in the requirements.txt file.

Bash

pip install -r requirements.txt
How to Run
Clone the repository:

Bash

git clone https://github.com/chauhanaman41/Infographio.git
cd Infographio
Set up the environment:

Bash

# Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate
Install dependencies:

Bash

pip install -r requirements.txt
Run the analysis and dashboard: Open and run the cells in the InfoGraphIQ_Project.ipynb notebook. The final cells will launch the interactive Plotly Dash dashboard.
